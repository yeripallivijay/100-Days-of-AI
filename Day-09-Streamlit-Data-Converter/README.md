# Streamlit Data Assistant (CSV / Excel Uploader)

A simple Streamlit web app that lets users upload CSV or Excel files and instantly preview the dataset in a clean table.

## Demo
- Live App: (https://yeripallivijay-data-cleaning-tool-app-97novd.streamlit.app/)
- GitHub Repo: https://github.com/yeripallivijay/<data_cleaning_tool>/

## Features
- Upload CSV (`.csv`) files
- Upload Excel (`.xlsx`) files
- Automatic file-type detection (reads CSV with `pandas.read_csv` and Excel with `pandas.read_excel`)
- Quick data preview inside the browser

## Tech Stack
- Python
- Streamlit
- Pandas
- OpenPyXL (for reading `.xlsx`)
