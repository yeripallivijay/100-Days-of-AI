# Day 22 – Streamlit for AI Applications

## Covered Concepts
- **Streamlit Basics:** `st.title()`, `st.sidebar`, `st.cache_data()`.
- **Data Visualization:** `st.dataframe()`, `st.pyplot()`, `st.plotly_chart()`.
- **Interactivity:** `st.slider()`, `st.file_uploader()`, `st.selectbox()`.
- **Deployment:** `streamlit hello` and sharing via Streamlit Cloud.

## Why it matters for AI?
- **Model Demos:** Upload an image → Get prediction → See confidence scores.
- **Data Exploration:** Interactive dashboards for stakeholders.
- **Prototyping:** Test your ML model with real user inputs before full deployment.
